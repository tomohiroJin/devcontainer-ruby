def apply_function(array, func)
  array.map { |element| func.call(element) }
end

    高階関数: 関数を引数として受け取る

    @param array [Array] 処理対象の配列@param func [Proc] 各要素に適用する関数@return [Array] 処理後の配列

    使用例:

    func = ->(x) { x * x }
functional.apply_function([1, 2, 3], func)
# => [1, 4, 9]
